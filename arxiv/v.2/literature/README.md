# v.2 Literature Index

This folder contains local source copies used for the `v.2` embodied robotics architecture notes.

## Files

- `we all are robots - EP.108.pdf`
  - Lukas Ziegler newsletter PDF used for current-industry context.
  - Useful for:
    - `1X` dexterous hand claims,
    - `Holiday Robotics` VLS stack mention,
    - broad industry snapshot across US, China, and Korea.

- `GR00T N1 - An Open Foundation Model for Generalist Humanoid Robots.pdf`
  - Direct source for a published dual-system humanoid robot architecture.

- `Hume - Introducing System-2 Thinking in Visual-Language-Action Model.pdf`
  - Direct source for explicit `System 2` reasoning plus reactive `System 1` control in embodied robotics.

- `AgiBot World Colosseo - A Large-scale Manipulation Platform for Scalable and Intelligent Embodied Systems.pdf`
  - Source for `AgiBot` data platform and `GO-1` generalist policy.

- `Language-Conditioned Robotic Manipulation with Fast and Slow Thinking.pdf`
  - Direct source for fast/slow thinking in robotic manipulation.

- `Architecture Is All You Need - Diversity-Enabled Sweet Spots for Robust Humanoid Locomotion.pdf`
  - Source for layered-timescale humanoid locomotion control.
